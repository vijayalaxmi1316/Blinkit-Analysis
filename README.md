# Blinkit Sales Analysis – Power BI Project
## Project Overview
This project presents an end-to-end sales and performance analysis of Blinkit using Power BI. The dashboard provides actionable insights into sales trends, customer behavior, product performance, and outlet efficiency. It is designed to help stakeholders make data-driven decisions related to inventory, pricing, and regional strategy.

## Objectives
Analyze overall sales performance and growth trends
Understand customer purchasing behavior
Identify top-performing and underperforming products
Compare performance across outlet sizes and locations
Provide interactive visualizations for business insights

## Dataset Description

The dataset includes the following key attributes:
Item Details: Item Name, Item Type, Item Fat Content
Sales Metrics: Total Sales, Average Sales, Number of Items Sold
Outlet Information: Outlet Type, Outlet Size, Outlet Location Type, Establishment Year
Ratings: Customer ratings for products and outlets

Note: Data was cleaned and transformed within Power BI using Power Query.

## Tools & Technologies Used
Power BI Desktop – Data modeling & dashboard creation
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Measures and KPIs

## Key KPIs & Measures

Total Sales
Average Sales per Item
Total Number of Items Sold
Average Customer Rating

## Dashboard Features

Sales Trend Analysis – Year-wise and outlet-wise sales performance
Category Analysis – Sales by item type and fat content
Outlet Performance – Comparison by outlet size and location
Top Products – Best-selling and least-selling items
Interactive Filters & Slicers – Outlet type, item category, and location

## Key Insights

Medium-sized outlets contribute the highest share of total sales
Low-fat items show strong customer preference in urban locations
Tier 1 cities outperform Tier 2 and Tier 3 in terms of revenue
Customer ratings positively correlate with higher sales volumes

## Conclusion

The Blinkit Sales Analysis project successfully transforms raw sales data into meaningful business insights through interactive Power BI dashboards. The analysis highlights key revenue drivers such as outlet size, location, and product categories. By leveraging these insights, Blinkit can optimize inventory planning, improve customer satisfaction, and focus on high-performing outlets and products. This project demonstrates strong skills in data analysis, visualization, and business intelligence, making it a valuable addition to a data analyst portfolio.
